### P2（一般）

- **本地合成 ID 无法自动合并编辑后的作品**：ID 包含标题和发布时间。
  - 位置：`BrowserService._local_video_id()`
  - 影响：标题或发布时间编辑后可能生成新记录，旧记录仍保留。
  - 建议：页面未来出现公开稳定 ID 时增加迁移；当前在文档和返回 source 中明确限制。

- **固定类名前缀仍受页面改版影响**：标题与指标定位使用 CSS module 的语义前缀。
  - 位置：`browser/extractors.py` 中 DOM 脚本
  - 影响：页面重命名后可能降为 `partial`。
  - 建议：后续增加多选择器诊断和脱敏结构版本识别。
